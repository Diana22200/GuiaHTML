¿Qué es HTML? Mapa conceptual

Versiones de HTML

Las versiones de HTML son:

HTML 2.0
Se publicó en 1995. HTML 2.0 No soportaba tablas y la declaración de <body>, <Html> y <head> era opcional.
HTML 3.2
Se publicó en 1997. HTML 3.2 incorporo applets de JAVA y texto que podía circular al rededor de las imagénes
HTML 4.01
Se publicó en 1999. A partir de esta versión la W3C decidio dedicarse a el desarrollo de XHTML. Sin embargo, debido a que la WHATWG realizo un borrador de HTML 5, la W3C retomo la actividad estandarizadora de HTML
HTML 5,HTML 5.1, HTML 5,2
Se publicó en 2014. En HTML 5 se incorporan nuevas etiquetas, la posibilidad de introducir más archivos multimedia, etc.

Enunciado y descripción de etiquetas HTML

  1. <br>:Realiza un salto de línea

  2. <footer> </footer>:Contiene el pie de página

  3. <header> </header>:Contiene el encabezado de la página

  4. <nav> </nav>:Contiene el menú de navegación

  5. <article> </article>:Contiene las piezas individuales de contenido

  6. <aside> </aside>:Representa una sección con contenido que esta sutilmente relacionado con el contenido principal pero no es fundamental.

  7. <i> </i>:Añade cursiva

  8. <b> </b>:Texto en negrita.

  9. <a> href="http://ejemplo.com/" </a>:Sirve para enlazar.

  10. <title> </title>:Es el que le da el nombre al documento.

  11. <!DOCTYPE html>:Permite al navegador entender cuál es la versión de html que se esta utilizando.

  12. <main> </main>:Contiene el contenido principal.

  13. <dl> </dl>:Contiene una lista descriptiva.

  14. <dt> </dt>:Contiene un término en una lista descriptiva

  15. <dd> </dd>:Contiene la descripción de un término en una lista descriptiva

  16. <ol> </ol>:Contiene listas ordenadas

  17. <ul> </ul>:Contiene listas desordenadas

  18. <ol> </ol>:Contiene los elementos de una lista

  19. <p> </p>:Es texto organizado en párrafos

  20. <input> </input>:Permite ingresar información en un formulario
